SPY Return Analysis: Arithmetic vs Log Returns

This project explores the statistical properties of the SPDR S&P 500 ETF Trust (SPY) by analyzing its historical price data. The main focus is on comparing Arithmetic Returns and Logarithmic Returns (Log Returns) and examining their impact on key risk metrics such as Variance and Standard Deviation.

The project provides practical insights into how financial analysts, quantitative researchers, and portfolio managers measure returns and assess market risk.
